#### 二 聊天功能接口

请求方式：POST

接口url:

示例：https://agentos.i-tudou.com/api-2137107453692846363/v1/api/agents/2137107453692846364/sessions/8a71f45f-791b-4597-b0e6-83eb784c13ed/messages

其中8a71f45f-791b-4597-b0e6-83eb784c13ed是会话id，可以通过新增会话接口获取

接口请求参数：

{"role":"human","chat\_area":{"normal":"验证码登录功能测试用例？"},"perception_stream": false}

|  **参数名称**  |  **参数类型**  |  **是否必填**  |  **参数说明**  |  **示例值**  |
| --- | --- | --- | --- | --- |
|  role  |  String  |  是  |  角色  |  human  |
|  chat\_area  |  Map<String,Object>  |  是  |  聊天区参数  |  "chat\_area":{"normal":"验证码登录功能测试用例？"}}  |
|  normal  |  String  |  是  |  用户提问  |  验证码登录功能测试用例？  |
|  perception_stream  |  Boolean  |  否  |  感知是否流式返回  |  false  |

header

|  **参数名称**  |  **参数类型**  |  **是否必填**  |  **参数说明**  |  **示例值**  |
| --- | --- | --- | --- | --- |
|  satoken  |  String  |  否  |  token  |   |
|  version  |  String  |  是  |  子版本  |  9，在智能体简历中查看  |
|  officeid  |  String  |  否  |  办公室id  |   |

输出格式：

|  **参数名称**  |  **参数类型**  |  **参数说明**  |  **示例值**  |
| --- | --- | --- | --- |
|  session\_id  |  String  |  会话id  |   |
|  message\_id  |  String  |  消息id  |   |
|  message\_type  |  String  |  消息类型  |   |
|  content\_type  |  String  |  内容格式  |   |
|  event\_status  |  String  |  事件状态  |   |
|  data  |  Map<String,Object>  |  返回数据状态  |   |
